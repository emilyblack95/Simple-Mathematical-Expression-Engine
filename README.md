# Simple-Mathematical-Expression-Engine
Takes a String expression, using the operators: "*", "/", "+", "-", with no spaces, and returns the result. Supports Doubles as well.

Example Input (without quotations):
"-100/10*2" <br />
"-5-5" <br />
"7*5/30" <br />
"2.5*2.5" <br />
"1+2-3" <br /> 
"1-2+3"
